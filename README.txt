REACTION MOTORSPORTS WEB APP

This is a working static web app.

How to use it:
1. Unzip this folder.
2. Upload the CONTENTS of the folder to your website.
3. The main file is index.html.
4. Keep the assets folder, app.js, data.js, and styles.css together.

This is not an App Store native app.
This is a website-based mobile app/prototype you can upload to your website and view on a phone.

Files:
- index.html
- styles.css
- app.js
- data.js
- assets/logos
- assets/photos
- assets/sponsors

What works:
- Home screen
- Events screen
- Event filters
- Event detail screens
- Schedule screen
- Gallery screen
- Sponsors screen
- Venue screen
- Bottom navigation
- Real uploaded logos/photos/sponsor art
